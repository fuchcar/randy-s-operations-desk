<h1 align="center">Randy's Operations Desk</h1>

> A portfolio piece — an interactive, sanitized demo. No real data.

The dry-witted raccoon assistant for solo creatives. A personal operations command center where one calm screen runs a freelancer's whole business — and Randy quietly drafts the busywork.

**▶ Live demo:** https://meet-randy-showcase.lovable.app

---

## What it is

A personal operations command center for a one-person creative business — the kind of single screen a working photographer, designer, or filmmaker actually wants open all day. Today's priorities, money, clients, deliverables, calendar, and a catch-all "Ask Randy" box, all in one place.

Randy is the host: a professional, dry-witted raccoon AI assistant who explains what each room shows and offers to do the next step. Every figure, client, and event in this demo is fictional.

## Two ways to explore

- **Demo mode** — a fully-seeded fictional business (Aspen Ridge Realty, Maple & Co Weddings, Northgate Builders, Harbor Light Inn). Browse the whole desk as if it were a real working day.
- **Explore mode** — every automation surface ("Hand to Randy", "Send invoice", "Mark paid", "Sync calendar", "Parse bank text") opens a tasteful "here's what would happen if this were wired to your real tools" popup in Randy's voice.

## Features

- **Today** — a smart "do this next" pick, a board-heat gauge, hard deadlines, and a shoot-prep checklist for the next job on the books.
- **Money** — a true-ledger view: **Available → Coming (certain) → Spendable**, a color-coded "Where it goes" donut (Need / Protect / Business / Giving), income-vs-outflow bars, a debt payoff tracker, and expandable detail accordions for recurring, bills, buckets, and incoming.
- **Clients** — a mini-CRM grid with relationships, job history, and a detail panel for each.
- **Deliverables** — every project in flight with an honest drag-to-set progress slider.
- **Calendar** — Day / Week / Month / List views, Comfortable/Compact density, animated day-detail popovers, and owner filter chips (You / Partner / Team).
- **Ask Randy** — a catch-all input for anything that doesn't fit a room.
- **Guided tour** — a coachmark walkthrough of every room, with minimize/resume so you can roam without losing your place.
- **"What's this?" helper** — right-click (or long-press) any card and Randy explains what it's showing, in plain English.

## Tech

- React 19 + TanStack Start (file-based routing, SSR-ready)
- TypeScript (strict)
- Tailwind CSS v4 with a custom warm-editorial dark theme
- Framer Motion for spring/stagger/hover animation
- Zustand for in-memory app state
- lucide-react icons
- Vite + Bun

All data is in-memory mock data. No backend, no accounts, no integrations.

## Note

This is a sanitized public demo. It contains no real personal, client, or financial data — every name, number, and event is fictional and exists only to make the interface feel lived-in.
